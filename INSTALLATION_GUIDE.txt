TROUBLESHOOTING INSTALLATION:
==============================

Problem: Can't find "Load unpacked" button
Solution: Make sure Developer mode is enabled (toggle in top-right)

Problem: Extension won't install
Solution:
   - Make sure you're selecting the FOLDER, not a file
   - Check that all files are present (manifest.json required)
   - Try restarting browser

Problem: Extension shows error immediately
Solution: Backend server isn't running yet - start it first

Problem: Backend server won't start
Solution:
   - Install Python 3: python.org/downloads
   - Install sqlcmd: aka.ms/sqlcmd
   - Check SQL credentials in backend_server.py

Problem: "Port already in use" error
Solution:
   - Another program is using port 8000
   - Use different port: python backend_server.py 9000
   - Update extension settings to: http://localhost:9000/api/waitlist

Problem: Extension shows no data
Solution:
   - Check backend server is running
   - Check settings (âš™ï¸) for correct URL
   - Try opening http://localhost:8000/api/waitlist in browser
   - Should show JSON data

Problem: Clicking appointments doesn't work
Solution:
   - Check LiveAccess URL in settings
   - Make sure LiveAccess is accessible from your computer
   - Check popup permissions (right-click extension â†’ Manage)

Problem: Badge not showing count
Solution:
   - Wait 5 minutes for auto-update
   - Click ğŸ”„ to force refresh
   - Check extension has necessary permissions
